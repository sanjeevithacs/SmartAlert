# SmartAlert — Form Non-Responder Reminder System

A small project containing a Jupyter notebook that implements a form non-responder reminder workflow.

## Files
- `Form Non-Responder Remainder System.ipynb` — main notebook with the implementation.

## Usage
1. Open the repository in VS Code or Jupyter Notebook.
2. Open `Form Non-Responder Remainder System.ipynb` and run the cells.
